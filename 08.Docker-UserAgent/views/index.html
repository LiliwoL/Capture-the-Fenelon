Bienvenue sur l'application User-Agent

Cette application a plusieurs pages qui nécessitent des User-Agents spécifiques pour être accessibles.

Les User-Agents à utiliser sont :

  • Page 1: MySpecialAgent/1.0 fenelon1

  • Page 3: FinalAgent/3.0 fenelon3
